\doxysubsection{Detailed Description}
Represents a part or component of a model. 

\mbox{\hyperlink{classModelPart}{Model\+Part}} instances are used to build a hierarchical tree structure representing components of a model. Each part can have multiple child parts. This class also provides functionalities to load visual representations from STL files and manage their appearance properties such as visibility and color.
